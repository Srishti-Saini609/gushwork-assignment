# Gushwork Frontend Assignment

A responsive web page built using **HTML, CSS, and JavaScript (Vanilla)**.

## Features

- Sticky Navbar with scroll animation
- Responsive design (Desktop, Tablet, Mobile)
- Premium Image Carousel
- Image zoom on hover
- Smooth animations and micro-interactions
- Mobile hamburger menu

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla)

## Responsiveness

Fully responsive across:
- Desktop
- Tablet
- Mobile
